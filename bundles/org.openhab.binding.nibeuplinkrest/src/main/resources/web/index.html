Use this page to authorize openHAB to get data from your Nibe Uplink account. First you need to login to api.nibeuplink.com (using your Nibe uplink username and password) and register your application.

The callback URL you should specify is:

${callbackURL}

Then you need to create a Nibe Uplink API Bridge in openHAB and configure it with the Identifier and Secret that is generated

These are the Bridges you have configured and can be authorized:

${accounts}
